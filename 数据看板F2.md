### 1.文字换行展示

在`chart.source()`里配置

```
{
        name: {
          tickCount: 1,
         formatter(val) {
            return val.split('').join('\n')
          }
        }
     }
     
```

### 2.点击滑动展示图表信息

```
chart.tooltip({
  showItemMarker: false,
  onShow: function onShow(ev) {
    const items = ev.items;
    items[0].name = null;
    items[0].name = items[0].title;
    items[0].value = '¥ ' + items[0].value;
  }
});
```
