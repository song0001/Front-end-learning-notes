# Markdown使用语法

###  1.[标题]一级标题-六级标题

语法:# 标题名称

# 一级标题

## 二级标题

### 三级标题

### 2.[列表]无序列表-有序列表

无序列表:- 名称 

- 无序列表

有序列表: 1. 名称

1. 有序列表
2. 我是有序列表

### 3.链接

语法:`[名称](链接地址)`

如:[百度一下](http://www.baidu.com)

### 4.图片

语法:![](链接地址)

如:![](C:\Users\song\Desktop\03-DOM与BOM\课程 6B\8-17 AJAX\error.jpg)
