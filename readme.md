## cms
> 内容管理系统. 一般是指网站中后台.用于提供网站展示页面的动态内容的增删改查操作.

+ 有些网站是模板,后台cms是一套代码,页面换了ui一键生成.这种后台cms一般是一套成熟的产品,可以是通用cms.
+ 有些网站是定制开发,后台cms不能满足页面的个性化需求,所以此时后台cms也需要定制开发.
+ http://demo.zving.com/
+ https://gitee.com/doramart/DoraCMS?_from=gitee_search
